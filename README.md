# Angular Homes Extended (Agents + Projects)

This project extends the Angular Homes tutorial into a **modular, scalable application** — evolving from a simple UI into a structured system.

---

## 🧠 System Focus

Building reusable patterns that scale across features and prepare for real-world apps.

---

## 📍 Current State

* Homes → Agents → Projects
* Modular architecture (pages / components / services / interfaces)
* Shared navigation (`HeaderComponent`)
* Live search filtering

---

## 🏗️ Architecture

```text
src/app/
├── pages/
├── components/
├── services/
├── interfaces/
```

## 🔑 Concepts

* Separation of concerns
* Reusable feature patterns

* Reactive UI
